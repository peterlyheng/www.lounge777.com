Comment les machines à sous fonctionnent-elles et que signifient les symboles ?

Pour commencer à jouer sur une machine à sous, cliquez simplement dessus et le jeu s'affichera automatiquement. Sur chaque machine à sous, vous trouverez 5 rouleaux avec différents symboles. En fonction de la disposition des symboles sur les rouleaux lorsqu'ils s'arrêtent, vous gagnez ou perdez des jetons. Ces formations sont appelées lignes de paiement. Dans le coin inférieur gauche, vous sélectionnez le nombre de lignes que vous souhaitez jouer. Selon le type de machines à sous, le nombre de lignes de paiement que vous pouvez jouer sera différent. Plus vous sélectionnez de lignes de paiement, plus vous devrez miser de jetons. Juste à côté du bouton qui vous permet de sélectionner les lignes de paiement, vous pouvez choisir le nombre de jetons que vous souhaitez miser sur chaque ligne. Ces deux éléments combinés forment le nombre de jetons que vous misez à chaque partie. Vous pouvez ajuster ce montant pour chaque partie. Vous pouvez également directement choisir de jouer le nombre maximum de jetons. Pour cela, cliquez simplement sur le bouton « MISE MAXIMALE ». Tous les gains sont comptés de gauche à droite sur les rouleaux qui se trouvent côte à côte. Les règles exactes et détaillées pour les parties gratuites, le symbole Scatter, les symboles bonus et le nombre de lignes de paiement sont disponibles sur le tableau des gains de chaque machine à sous. Voici quelques informations générales :

Symbole Scatter

Le symbole Scatter peut apparaître n'importe où sur les rouleaux. En fonction de la machine à sous, vous remporterez une certaine quantité de parties gratuites avec un certain nombre de symboles Scatter. Dans certains cas, les rouleaux sur lesquels les symboles apparaissent comptent également (par exemple, 9 symboles Scatter sur les rouleaux 2 à 4). Vous trouverez les conditions à respecter pour les différents types de gains en cliquant sur « i » dans le coin inférieur gauche de la machine à sous. Tandis que tous les symboles comptent uniquement s'ils sont dans le bon ordre sur les lignes de paiement, les symboles Scatter peuvent également être gagnants sur d'autres rouleaux.

Lignes de paiement

Au début de la partie, vous sélectionnez le nombre de lignes de paiement que vous souhaitez jouer et le nombre de jetons que vous souhaitez miser. Seul le gain le plus élevé par ligne compte. S'il y a plusieurs gains sur une même ligne, ceux-ci sont additionnés. Le gain est crédité sur votre solde dès qu'il est accepté ou que l'option de mise est terminée.

Symbole bonus

Les symboles bonus peuvent déclencher des parties gratuites, voire même des gains spéciaux. Pour cela, un certain nombre de symboles bonus doivent apparaître sur les rouleaux en même temps. Consultez le tableau des gains de chaque machine à sous pour connaître les symboles qui sont considérés comme des symboles bonus, le nombre de symboles bonus nécessaires pour gagner et les autres fonctions spéciales.

Symbole Wild

Le symbole Wild est un joker, il apparaît sur la plupart des machines à sous. En tant que joker, le symbole Wild peut remplacer la plupart des autres symboles dans un jeu et permet donc d'obtenir tout un tas de combinaisons gagnantes. Le seul symbole que le joker ne peut habituellement pas remplacer est le symbole bonus. Consultez la section d'aide du jeu respectif pour savoir quel symbole est un joker Wild pour ce jeu et quels symboles il peut remplacer.

Risque

Certaines machines à sous vous permettent de parier avec vos gains. Vous pouvez repartir avec vos gains ou décider de les parier, ce qui peut vous permettre de gagner encore plus de jetons, mais aussi d'en perdre. Après avoir gagné, appuyez simplement sur « Risque » si vous souhaitez parier vos gains. Afin de doubler vos gains, sélectionnez une couleur de carte : rouge ou noire. Si vous faites le bon choix, vos gains seront doublés. Si vous sélectionnez le bon symbole de carte (cœur, carreau, trèfle, pique), vous pouvez même quadrupler vos gains. Vous pouvez parier avec vos gains jusqu'à 5 fois d'affilée.

Parties gratuites

Les parties gratuites peuvent être déclenchées, par exemple, par l'apparition simultanée de trois symboles bonus ou plus. Au cours de ces parties gratuites, vous pouvez gagner les mêmes choses que pendant des parties normales. Le nombre de lignes et votre mise par ligne restent les mêmes qu'avant l'activation des parties gratuites. Cliquez sur le symbole « i » (ou sur le tableau des gains correspondant) dans le coin inférieur gauche pour connaître les symboles qui déclenchent des parties gratuites, le nombre de parties gratuites déclenchées et des informations détaillées sur d'autres fonctions spéciales.
